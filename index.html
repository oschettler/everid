What is this?


It uses the free Github Pages as hosting platform. This means, the resulting site (layout, configuration and content) lives in one of your Git repositories at Github and can be edited to your liking. No lock-in!

EverID currently allows you to choose one of the themes from Bootswatch to style your site. 

How it works

EverID is written as an open source application in PHP. It uses an Sqlite database to make it easy to host the software yourself. 

But you don’t have to. At everid.net you can log-in with your Evernote credentials, using OAuth. This means we don’t get your Evernote username and password, but only an authentication token that you can revoke at any time.

The same holds for the Github site. We use OAuth there as well and the Github API to write the pages of your site to a branch gh-pages in the Git repository you have configured. From this, Github uses their Jekyll software to create a static website on their Pages facility.
